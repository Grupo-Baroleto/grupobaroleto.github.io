# 🌐 Grupo Baroleto — Site Institucional

> Site oficial do **Grupo Baroleto**, uma holding open source dedicada à inovação tecnológica, sustentabilidade e liberdade digital.

🔗 **Acesse:** [grupobaroleto.github.io](https://grupobaroleto.github.io)

---

## 📋 Sobre este repositório

Este repositório contém o código-fonte do site institucional do Grupo Baroleto, publicado via **GitHub Pages**. O site é estático, de arquivo único (`index.html`), sem dependências de frameworks ou build tools — roda diretamente no navegador.

---

## 🏢 Sobre o Grupo Baroleto

O **Grupo Baroleto** nasceu em novembro de 2025, com sede em Palmas, Tocantins, com a missão de unir inovação tecnológica, sustentabilidade e filosofia open source. Nossa visão é levar a marca Baroleto a todos os ramos do mercado — do local ao internacional.

**Frentes de atuação:**

| Empresa | Área |
|---|---|
| Baroleto Smart Systems | Automação residencial, empresarial e industrial |
| Baroleto Agro | Sustentabilidade, energia renovável e agricultura inteligente |
| Baroleto Foods | Alimentação saudável e produção artesanal |
| Baroleto Cloud | Soluções digitais e IoT open source |
| Baroleto Services | Serviços tecnológicos e administrativos |

## 🛠️ Tecnologias utilizadas

- **HTML5** semântico
- **CSS3** com variáveis, animações e design responsivo
- **JavaScript** vanilla — scroll reveal, navegação e formulário de contato
- **Google Fonts** — Cinzel, Cormorant Garamond, DM Sans
- **Formspree** — backend de formulário sem exposição de email
- **GitHub Pages** — hospedagem estática gratuita

---

## 🚀 Como publicar alterações

1. Clone o repositório:
   ```bash
   git clone https://github.com/grupobaroleto/grupobaroleto.github.io
   ```

2. Faça suas alterações no `index.html`

3. Commit e push:
   ```bash
   git add .
   git commit -m "descrição da alteração"
   git push
   ```

4. O GitHub Pages publica automaticamente em até 2 minutos em [grupobaroleto.github.io](https://grupobaroleto.github.io)
